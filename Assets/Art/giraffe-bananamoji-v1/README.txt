https://timswast.itch.io/giraffe-bananamoji

Web version:
https://stickers.bananajuice.tech/en/collection/giraffe/


Discord
-------

You must be a server admin to upload emoji to Discord.

See: https://support.discordapp.com/hc/en-us/articles/207619737-Adding-Emoji-Magic

1.  In the server settings, go to the Emoji tab.
2.  Click the upload Emoji button.
3.  Select the emoji images to upload.


Slack
-----

To upload emoji in bulk to Slack:

1.  Install the Neutral Face Emoji Tools Chrome Extension.
    https://chrome.google.com/webstore/detail/neutral-face-emoji-tools/anchoacphlfbdomdlomnbbfhcmcdmjej?hl=en

2.  Customize your Slack workspace.
    https://your-slack-instance.slack.com/customize/emoji

3.  Drag and drop all the emoji images from the slack/ directory to the "Bulk
    Emoji Uploader", provided by the Neutral Face Emoji Tools.
